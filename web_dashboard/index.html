Why PropSync?

Everything you need to manage
your trading infrastructure

Real-Time Telemetry

Monitor equity, floating P/L, drawdown, and execution stats with automatic updates from MT5.

Multi-Node Copytrade Network

Connect a master account with multiple slaves with configurable risk ratios and automatic trade replication.

Prop Firm Tracking

Manage your evaluated accounts with profit target progress bars and drawdown alerts for Phase 1, Phase 2, and Funded.

Cloud & Local Dashboard

Access your metrics from any device via the cloud version (Netlify) or from your desktop terminal with pywebview.

Risk Control

Automatic calculation of recommended risk multiplier between accounts, with visual alerts when drawdown exceeds 85%.

Supabase Sync

Trade history, configurations, and metrics synced to the cloud with your personal free Supabase account.

How does it work?

Setup and configuration
in under 10 minutes

01

Download from GitHub

Clone the repository and create a Python virtual environment. Install dependencies with pip install -r requirements.txt.

→
02

Configure Your Accounts

Enter the MT5 credentials of your master and slave nodes from the Network Topology panel.

→
03

Start the RPA Engine

Run python main.py to launch the local interface. The engine connects to MT5 and starts telemetry and auto-copying.

→
04

Monitor from Anywhere

Access the cloud dashboard from your mobile device or browser to review real-time metrics and performance.
